Pixel Dungeon

A roguelike adventure into the depths

About the Game

Pixel Dungeon is a traditional roguelike game with pixel-art graphics and a simple interface. Every game is a unique challenge with randomly generated levels, enemies, and items. Explore the dungeon, collect treasures, and try to survive as long as you can.

The game features permadeath, meaning when you die, you start over from the beginning. But don't worry—each run teaches you something new about the dungeon's secrets.

Key Features

  • Procedurally generated dungeons
  • Four unique character classes
  • Over 100 items to discover
  • Challenging boss battles
  • Pixel art aesthetic
